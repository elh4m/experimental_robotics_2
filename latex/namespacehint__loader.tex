This function is a callback function of \textquotesingle{}/oracle\+\_\+service\textquotesingle{} node. It waits for \textquotesingle{}/oracle\+\_\+service\textquotesingle{} service request from the \textquotesingle{}motion\+\_\+controller\textquotesingle{} node. Based on the type of recieved request, it loads the hint in the ARMOR reasonser, start the reasoner to deduced a hypotheses based on previously loaded hints and request ARMOR reasoner for the list of \textquotesingle{}COMPLETE\textquotesingle{} hypotheses. If the hypotheses is \textquotesingle{}CONSISTENT\textquotesingle{} it respond with \textquotesingle{}True\textquotesingle{} otherwise it respond with \textquotesingle{}False\textquotesingle{}. Similarly if the hypotheses is \textquotesingle{}CONSISTENT\textquotesingle{} then the user request this node to check if the hypotheses is also \textquotesingle{}CORRECT\textquotesingle{}. A hypotheses correct if it is \textquotesingle{}CONSISTENT\textquotesingle{} and belong to a predefined list of correct hypotheses in this node.
