Based on the request recieved, it loads the hint in the ARMOR reasonser, start the reasoner to deduced a hypotheses based on the previously loaded hints and request ARMOR reasoner for the list of \textquotesingle{}COMPLETE\textquotesingle{} hypotheses. If the recently deduced hypothesis is \textquotesingle{}COMPLETE\textquotesingle{} then it checks its \textquotesingle{}CORRECTNESS\textquotesingle{}. ~\newline
